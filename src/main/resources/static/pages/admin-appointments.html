How to use this page

This view is a hospital-wide schedule: every row ties a patient to a doctor, department, date, time, and status. Use it for daily operations checks and historical audits.

  • Scan Date and Time to see load by day; pair with doctor lists if you need to balance clinics.
  • Status reflects workflow state (e.g. scheduled, completed)—use it to spot stalled or overdue visits.
  • Patient and doctor names help reconcile this table with user and doctor admin screens when correcting records.
  • New bookings and changes originate from the patient booking flow; this page is the consolidated read-out for admins.
