How to use this page

Each doctor is tied to a department and appears in patient booking flows. Keep names, emails, and specialization current so the right clinician is always shown.

  • Add Doctor opens a form: password is required for new accounts; for edits, leave it blank to keep the existing password.
  • Set Department from the dropdown—if a specialty is missing, add it under Departments first.
  • Use Specialization and Availability text to help patients understand who to book and when slots are offered.
  • Delete removes the doctor permanently—confirm there are no outstanding dependencies if your hospital policy requires it.
