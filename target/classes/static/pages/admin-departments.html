How to use this page

Departments are the backbone of routing: patients pick a specialty, then doctors in that unit. Clear names and short descriptions reduce confusion at booking time.

  • Use Add Department for new units; prefer consistent naming (e.g. “Cardiology” not mixed abbreviations).
  • Description is optional but useful on public-facing screens and for internal reporting labels.
  • Edit updates the record in place; doctors already assigned stay linked—verify names still match your org chart.
  • Delete should be used carefully if doctors or appointments still reference the department; fix assignments first if your workflow requires it.
